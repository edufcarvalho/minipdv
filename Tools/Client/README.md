# Bruno API Client Setup

This is a Bruno API Client setup, it can be used to do do requests to the API more easily

## How to Install and use Bruno

### Installation

#### VSCode Extension
You can use Bruno inside `Visual Studio Code` easily with [VSCode Marketplace](https://marketplace.visualstudio.com/items?itemName=bruno-api-client.bruno)

#### Standalone
You can install Bruno at [Bruno Download Page](https://www.usebruno.com/downloads), download the executable for your architecture

### Execution
After installing, open Bruno, click the plus icon, click `Open collection`, and point Bruno to the `Tools\Client` folder in order to access Bruno requests
